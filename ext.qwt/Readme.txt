Notes
=====

This vle package is a distribution of the Qwt Library
(http://qwt.sourceforge.net/) developped by Uwe Rathmann and Josef Wilgen.
We thank them for this work.

Requirements
============

vle      1.3  http://www.vle-project.org
boost    1.49 http://www.boost.org
cmake    2.8  http://www.cmake.org


Installation
============

$ vle --package=ext.qwt configure
$ vle --package=ext.qwt build
